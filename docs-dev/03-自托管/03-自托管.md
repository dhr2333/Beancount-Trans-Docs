<summary>🤔 为什么需要设置默认分类？</summary>

Beancount-Trans 在处理无法自动识别分类的交易时，会将其归入默认分类。
设置一个合理的默认分类（如 `其他支出`）可以避免报表中出现“未分类”条目，保持账目整洁。

!!! info "关于数据迁移"
    迁移过程只会读取原始账单文件，**不会**修改或删除你的任何本地数据。
