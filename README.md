# Beancount-Trans 文档

[Beancount-Trans](https://github.com/dhr2333/Beancount-Trans) 的文档站点，使用 Docusaurus 构建。

📖 **在线地址**: [https://trans.dhr2333.cn/docs/](https://trans.dhr2333.cn/docs/)

## 本地开发

### 环境要求

- Node.js 16.14+
- npm / yarn

### 常用命令

```bash
# 安装依赖
npm install

# 启动开发服务器（支持热重载）
npm run dev

# 构建静态站点
npm run build
```
